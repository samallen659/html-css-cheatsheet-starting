CSS Selectors

Basic Selectors

Selector Name Description
p Type Selector Selects all html elements of type p
.example Class Selector Selects all html with the class "example" applied to it
#example ID Selector Selects the html element with the id of "example"

Attribute Selectors

Selector Description
[attr] Selects elements with an attribute name of attr.
[attr=value] Selects elements with an attribute name of attr whose value is exactly value
[attr~=value] Selects elements with an attribute name of attr whose value is a whitespace seperated list of words, one of which is exactly value
[attr|=value] Selects elements with an attribute name of attr whose value can be exactly value or can begin with value immediately followed by a hyphen
[attr^=value] Selects elements with an attribute name of attr whose value is prefixed by value
[attr$=value] Selects elements with an attribute name of attr whose value is suffixed by value
[attr*=value] Selects elements with an attribute name of attr whose value contains at least one occurence of value within the string
[attr operator value i] Adding an i (or I) before the closing bracket causes the value to be compared case-insensitively (for characters within the ASCII range).
[attr operator value s] Adding an i (or I) before the closing bracket causes the value to be compared case-insensitively (for characters within the ASCII range).

Selector Combinators

Selector Name Description
div p Descendant Combinator Selects all p elements that are inside a div
div > p Child Combinator Selects all p elements that are nested directly inside a div element
div ~ p General Sibling Combinator Selects all p elements that follow a div, immediately or not
div + p Adjacent Sibling Combinator Selects all p elements that immediately follow a div element
col || td Column combinator Selects all td elements that belong to the scope of the col
